# Example — Drip&Co coffee shop

A complete miniature Adloom project you can run in 5 minutes.

| File | What it shows |
|------|---------------|
| `config.json` | brand palette, grid, ratios, hashtags, schedule |
| `brand-voice.md` | output shape of `/adloom-voice` |
| `captions.md` | 3-day campaign copy in that voice |
| `plate_4x5.png` | AI plate generated by `scripts/gen.mjs` (prompt below) |
| `demo_4x5.html` | strict-grid artboard compositing text/CTA/logo over the plate |
| `demo_4x5.png` | final render via `scripts/render.mjs` |
| `concept_battery_4x5.png` | concept plate from `/adloom-concept` (route 2: UI in the real world — a phone battery built from coffee) |
| `concept_battery_4x5.html` | poster-mode artboard: editorial type scale-contrast + info-strip footer |
| `concept_battery_final_4x5.png` | the finished concept ad |

## Reproduce it

```bash
# 1. plate (needs GKEY in .env)
node scripts/gen.mjs gemini-3-pro-image-preview examples/coffee-shop/plate_4x5.png "4:5" \
  "Premium cinematic advertising photograph for a specialty coffee brand. A barista's hands pouring \
latte art into a ceramic cup on a warm wooden counter, soft morning window light, shallow depth of \
field, steam rising, deep forest-green #1F3A33 and cream tones, cozy premium cafe atmosphere. \
Generous soft empty space in the TOP THIRD for a headline. Vertical 4:5, hyper realistic, no text, no logos."

# 2. extend the master to the other ratios (same scene, not a re-roll)
node scripts/gen_edit.mjs examples/coffee-shop/plate_4x5.png examples/coffee-shop/plate_1x1.png "1:1" \
  "Reframe this EXACT same photo into 1:1. Same subject, same scale, centered; extend the background sideways. Seamless."

# 3. composite + render
node scripts/render.mjs examples/coffee-shop/demo_4x5.html examples/coffee-shop/demo_4x5.png 1080 1350
```

Then run `/adloom-review` on the output, and `/adloom-schedule` when it passes.
